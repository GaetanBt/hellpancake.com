À propos de ce site

Ce site est construit grâce au Générateur de Site Statique Eleventy et est hébergé sur GitHub Pages. Tout ce que vous retrouverez ici est transparent, open source et le code source est disponible sur GitHub. Aucune donnée personnelle n'est collectée d'une quelconque manière via ce site, et votre vie privée est respectée.
